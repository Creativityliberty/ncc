# Bases scientifiques NCC-V0

Cette V0 s’appuie sur un filtrage critique des sources. On ne copie pas les cadres existants ; on garde ce qui renforce le noyau.

## Sources socle

[S1] CoALA — utile pour situer mémoire, action et décision dans les agents de langage.

[S2] Memory Networks — utile pour justifier une mémoire lisible/inscriptible couplée à l’inférence.

[S3] Missing Knowledge Layer — utile pour séparer mémoire, connaissance, sagesse/politique et inférence temporaire.

[S4] Atangana & Baleanu — utile pour penser une mémoire non locale et non singulière.

[S5] Atangana fractal-fractional — utile pour penser une mémoire multi-échelle.

[S6] Toufik & Atangana — utile pour le passage futur vers des approximations numériques.

[S7] Mamba — utile pour la propagation/oubli sélectif de l’information dans les longues séquences.

[S8] Liquid Time-Constant Networks — utile pour les dynamiques d’état bornées/adaptatives.

[S9] Neuro-symbolic AI — utile pour le raisonnement, les règles, la vérification et la métacognition.

[S10] Datasheets for Datasets — utile pour documenter la provenance des traces NCC.

## Position critique

NCC-V0 ne prétend pas prouver une AGI. Il fournit un banc d’essai local pour tester un cycle cognitif borné : observation, intention, écart, transformation, mémoire, action et feedback.
